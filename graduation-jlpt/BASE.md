# 졸업JLPT

어플명: 졸업JLPT

"words" 디렉토리에 각각의 json파일이 있습니다

json 파일의 구조는 다음과 같습니다

JLPT N1 단어 모음
words/jlpt_1.json

JLPT N2 단어 모음
words/jlpt_2.json

JLPT N3 단어 모음
words/jlpt_3.json

JLPT N4 단어 모음
words/jlpt_4.json

JLPT N5 단어 모음
words/jlpt_5.json

이거를 내장디비를 사용해서 외우도록 하고싶습니다.

어플은 다음과 같은 기능이 필요합니다

JLPT N5, N4, N3, N2, N1 순으로 선택하는 리스트

선택하면 20개씩 이미 나눠져있고 Unit 을 선택하는 리스트

Unit을 선택하면 가운데에 한자가 나오고
밑에 버튼에 "한국어" 클릭하면 한자 밑에 뜻 출력되고
밑에 버튼에 "히라가나" 클릭하면 한자 위에 히라가나 출력되고
알고있는지 외운건지 버튼 만들어서 기록하고싶어
